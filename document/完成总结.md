## JSON Schema 支持的题型 (11 种)

| 题型 | rows_to_answer | 说明 |
|------|----------------|------|
| SINGLE_CHOICE | 1 | 单选题（4个选项） |
| MULTIPLE_CHOICE | 1 | 多选题（≥2个选项） |
| TRUE_FALSE | 1 | 判断题 |
| FILL_BLANK | 1 | 填空题 |
| CALCULATION | 3 | 计算题 |
| WORD_PROBLEM | 3 | 应用题 |
| GEOMETRY | 3 | 几何题 |
| POETRY_APP | 3 | 古诗文鉴赏 |
| ESSAY | 5 | 作文题 |
| READ_COMP | 动态 | 阅读理解 |
| CLOZE | 动态 | 完形填空 |

---

## 新旧接口对比

| 特性 | 旧接口 `/api/questions/generate` | 新接口 `/api/questions/structured` |
|------|--------------------------------|----------------------------------|
| 输出格式 | Markdown | JSON |
| 前端渲染 | 通用 Markdown 渲染 | 题型组件渲染 |
| 答案显示 | Markdown 中 separate | 不返回答案 |
| 预留行数 | 不支持 | 自动计算填充 |
| 老数据兼容 | - | 不兼容 |

## 使用方法

### 1. 安装依赖
```bash
cd backend
pip install jsonschema
```

### 2. 启动服务
```bash
# 后端
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd frontend
npm run dev
```

### 3. 访问测试页面
浏览器打开：`http://localhost:5173/structured`
