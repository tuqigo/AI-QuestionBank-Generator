# 结构化题目功能说明

## 功能简介

结构化题目功能允许 AI 输出符合 JSON Schema 的结构化数据，前端根据题型使用对应组件进行渲染。

## 快速开始

### 1. 后端安装依赖

```bash
cd backend
pip install jsonschema
```

### 2. 启动后端

```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 3. 启动前端

```bash
cd frontend
npm run dev
```

### 4. 测试新接口

访问 `http://localhost:5173/structured` 查看测试页面。

## 新 API 接口

### `/api/questions/structured` (POST)

**请求体:**
```json
{
  "prompt": "出1道七年级数学单选题"
}
```

**响应示例:**
```json
{
  "meta": {
    "subject": "math",
    "grade": "grade7",
    "title": "初一数学综合练习题"
  },
  "questions": [
    {
      "type": "SINGLE_CHOICE",
      "stem": "下列各数中，属于负有理数的是",
      "options": ["A. $3$", "B. $0$", "C. $-\\sqrt{2}$", "D. $-0.5$"],
      "knowledge_points": ["有理数的概念"],
      "rows_to_answer": 1
    }
  ],
  "record_id": 123,
  "short_id": "abc123",
  "created_at": "2026-03-13T08:00:00Z"
}
```

## 题型支持

| 题型 | rows_to_answer | 说明 |
|------|----------------|------|
| SINGLE_CHOICE | 1 | 单选题 |
| MULTIPLE_CHOICE | 1 | 多选题 |
| TRUE_FALSE | 1 | 判断题 |
| FILL_BLANK | 1 | 填空题 |
| CALCULATION | 3 | 计算题 |
| WORD_PROBLEM | 3 | 应用题 |
| GEOMETRY | 3 | 几何题 |
| POETRY_APP | 3 | 古诗文鉴赏 |
| ESSAY | 5 | 作文题 |
| READ_COMP | 动态 | 阅读理解（题干行数 + 子题数 × 2） |
| CLOZE | 动态 | 完形填空（题干行数 + 子题数 × 2） |

## AI 提示词调整

修改 `backend/config.py` 中的 `QUESTION_SYSTEM_PROMPT`：

```
【输出要求】
- 只输出完整、可直接解析的 JSON
- 不要包含 markdown 标记、解释文字、emoji
- 答案不要输出（不需要"## 答案"部分）
- 所有字段严格按照 Schema 规定的类型输出
```

## 与旧接口对比

| 特性 | 旧接口 `/generate` | 新接口 `/structured` |
|------|-------------------|---------------------|
| 输出格式 | Markdown | JSON |
| 前端渲染 | 通用 Markdown | 题型组件 |
| 答案显示 | 需要分割 | 不返回答案 |
| 题目数量控制 | 在提示词中指定 | 在提示词中指定 |
| 预留行数 | 不支持 | 自动计算填充 |

### JSON 校验失败处理
如果 AI 返回的 JSON 不符合 Schema，后端会返回 502 错误：
```json
{
  "detail": "JSON 校验失败: 字段 'type': ..."
}
```

## 常见问题

### Q: AI 返回的 JSON 格式不正确？
A: 检查 `backend/config.py` 中的 System Prompt，确保要求 AI 只输出 JSON。

### Q: 如何添加新的题型？
A: 1. 在 `question_schema.json` 中添加题型定义；2. 在 `structured_question.py` 中添加 Pydantic 模型；3. 在 `QuestionRenderer.tsx` 中添加渲染逻辑。

### Q: 前端如何调用新接口？
A: 使用 `generateStructuredQuestions(prompt)` 函数，参考 `StructuredPreview.tsx`。
