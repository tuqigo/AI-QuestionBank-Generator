# 题目模板配置需求实现步骤

本文档总结了一类常见需求的实现步骤：**配置或更新题目模板记录，使其能够按照指定要求生成题目**。

典型需求示例：
> "查询 question_templates 的 id 为 14 的记录，理解其题目要求（年级、知识点、name、description），填充 template_pattern、variables_config 和 generator_module 字段，使模板能正常生成题目。"

---

## 实现步骤

### 步骤 1：查询并理解模板记录

使用 DbHelper 查询模板记录，理解其要求：

```python
from utils.db_helper import DbHelper

# 查询模板记录
template = DbHelper.get_template(14)
print(f"name: {template['name']}")
print(f"grade: {template['grade']}")
print(f"subject: {template['subject']}")
print(f"question_type: {template['question_type']}")
print(f"example: {template['example']}")
print(f"knowledge_point_id: {template['knowledge_point_id']}")
print(f"description: {template['description']}")

# 查询关联的知识点
if template['knowledge_point_id']:
    kp = DbHelper.get_knowledge_point(template['knowledge_point_id'])
    print(f"知识点：{kp['name']}")
```

**需要理解的要素**：
- **年级** (`grade`)：如 `grade2` 表示二年级
- **学科** (`subject`)：如 `math` 表示数学
- **题目类型** (`question_type`)：如 `ORAL_CALCULATION` 表示口算题
- **示例题目** (`example`)：如 `["26+12"]` 提示题目格式
- **知识点** (`knowledge_point_id` → `knowledge_points.name`)：如 "100 以内的加法和减法（二）"
- **模板名称和描述** (`name`, `description`)：如 "两位数加两位数（不进位）"

---

### 步骤 2：检查现有生成器

查看 `backend/services/template/generators/__init__.py` 中已注册的生成器：

```python
# 已注册的生成器
GENERATOR_REGISTRY = {
    "fraction_arithmetic_comparison": FractionArithmeticComparisonGenerator,
    "mixed_addition_subtraction": MixedAdditionSubtractionGenerator,  # 加减法统一生成器

    "vertical_addition_subtraction": VerticalAdditionSubtractionGenerator,
    "decimal_arithmetic": DecimalArithmeticGenerator,
    "unit_conversion_comprehensive": UnitConversionComprehensiveGenerator,

```

**选择生成器的原则**：
1. **优先复用现有生成器**：不要急于创建新生成器
2. **检查生成器是否支持所需约束**：如 `ensure_no_carrying`（不进位）
3. **如需要，扩展现有生成器**：添加新的配置参数和约束逻辑

---

### 步骤 3：扩展生成器（如需要）

如果现有生成器不支持所需约束，需要扩展它。

**示例：为加减法生成器添加 `ensure_no_carrying` 约束**

编辑 `backend/services/template/generators/mixed_addition_subtraction.py`：

```python
# 1. 在 generate() 方法中读取新配置参数
ensure_no_carrying = template_config.get("ensure_no_carrying", False)  # 加法不进位

# 2. 在 simple 题型中添加约束检查
if q_type == "simple":
    a = random.randint(num_min, num_max)
    b = random.randint(num_min, num_max)
    op = random.choice(op_values)

    # 检查不进位约束（仅加法）
    if ensure_no_carrying and op == "+":
        # 个位相加不能进位
        if (a % 10) + (b % 10) >= 10:
            continue
        # 十位相加不能进位
        if (a // 10) + (b // 10) >= 10:
            continue

    result = a + b if op == "+" else a - b
    stem = f"{a}{op}{b}=[BLANK]"
```

**如需添加新的知识点**，更新 `get_knowledge_points()` 方法：

```python
def get_knowledge_points(self, template_config: dict) -> List[str]:
    points = ["百以内加减法"]
    if template_config.get("ensure_no_carrying", False):
        points.append("两位数加两位数（不进位）")
    return points
```

### 步骤 4：确定要填充的字段

需要填充的字段：

| 字段 | 说明 | 示例值 |
|------|------|--------|
| `template_pattern` | 模板模式字符串 | `{a} + {b} = [BLANK]` |
| `variables_config` | JSON 配置 | 见下方 |
| `generator_module` | 生成器模块名 | `mixed_addition_subtraction` |
| `is_active` | 是否启用 | `1` |

**variables_config 配置示例**（两位数加两位数不进位）：

```json
{
  "num": {"min": 10, "max": 99},
  "op": {"values": ["+"]},
  "question_complexity": ["simple"],
  "ensure_no_carrying": true,
  "ensure_sum_within_100": true,
  "rendering_config": {
    "layout": "multi",
    "columns": 4,
    "font_size": 18,
    "answer_width": 32,
    "answer_style": "blank",
    "show_question_number": true
  }
}
```

### 步骤 6：测试生成器

验证生成器是否正常工作：

```python
import sys
sys.path.insert(0, 'backend')

from services.template.generators.mixed_addition_subtraction import MixedAdditionSubtractionGenerator

generator = MixedAdditionSubtractionGenerator()

template_config = {
    "num": {"min": 10, "max": 99},
    "op": {"values": ["+"]},
    "question_complexity": ["simple"],
    "ensure_no_carrying": True,
    "ensure_sum_within_100": True,
}

questions = generator.generate(template_config, quantity=10, question_type="ORAL_CALCULATION")

print("=== 生成的题目 ===")
for i, q in enumerate(questions, 1):
    print(f"{i}. {q['stem']}")
    # 验证：60+21=[BLANK] (不进位)
```

### 生成器选择指南

| 题目类型 | 推荐生成器 | 配置要点 |
|----------|------------|----------|
| 简单加减法 | `mixed_addition_subtraction` | `num` 范围，`op` 运算符，`ensure_positive` |
| 不进位加法 | `mixed_addition_subtraction` | 添加 `ensure_no_carrying: true` |
| 连加连减 | `mixed_addition_subtraction` | `question_complexity: ["consecutive_add"]` |
| 加减混合 | `mixed_addition_subtraction` | `question_complexity: ["mixed_operation"]` |
| 乘法除法 | `multiplication_division_comprehensive` | `factor`, `divisor`, `quotient` 范围 |
| 单位换算 | `unit_conversion_comprehensive` | `unit_category`, `convert_types` |
| 分数比较 | `fraction_arithmetic_comparison` | `denominator`, `numerator` 范围 |
| 竖式填空 | `vertical_addition_subtraction` | `blank_positions`, `operation_types` |

---

## 注意事项

1. **先查后改**：先查询理解记录，再进行更新
2. **复用优先**：优先复用现有生成器，不要急于创建新的
3. **约束验证**：扩展生成器后，务必测试约束是否生效
4. **JSON 格式**：`variables_config` 是 JSON 字符串，使用 `DbHelper.update_json_column` 自动序列化
5. **知识点关联**：确保 `knowledge_point_id` 关联的知识点存在且匹配

## 附录：完整脚本示例

```python
"""
模板配置脚本示例：配置 ID=14 模板（两位数加两位数不进位）
"""
import sys
sys.path.insert(0, 'backend')

from utils.db_helper import DbHelper
from services.template.generators.mixed_addition_subtraction import MixedAdditionSubtractionGenerator

# ==================== 1. 查询并理解模板 ====================
template = DbHelper.get_template(14)
print(f"模板名称：{template['name']}")
print(f"年级：{template['grade']}")
print(f"知识点 ID: {template['knowledge_point_id']}")

# 查询知识点
kp = DbHelper.get_knowledge_point(template['knowledge_point_id'])
print(f"知识点：{kp['name']}")

# ==================== 2. 更新模板配置 ====================
DbHelper.update_template(
    14,
    template_pattern="{a} + {b} = [BLANK]",
    generator_module="mixed_addition_subtraction",
    is_active=1
)

DbHelper.update_template_variables(
    14,
    {
        "num": {"min": 10, "max": 99},
        "op": {"values": ["+"]},
        "question_complexity": ["simple"],
        "ensure_no_carrying": True,
        "ensure_sum_within_100": True,
        "rendering_config": {
            "layout": "multi",
            "columns": 4,
            "font_size": 18,
            "answer_width": 32,
            "answer_style": "blank",
            "show_question_number": True
        }
    }
)

# ==================== 3. 测试生成 ====================
generator = MixedAdditionSubtractionGenerator()
config = {
    "num": {"min": 10, "max": 99},
    "op": {"values": ["+"]},
    "question_complexity": ["simple"],
    "ensure_no_carrying": True,
}

questions = generator.generate(config, 5, "ORAL_CALCULATION")
print("\n=== 生成的题目 ===")
for q in questions:
    print(q['stem'])

print("\n配置完成！")
```
